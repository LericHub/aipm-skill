# Brainstorm - 积分计算器应用

## 项目背景
- 项目类型：天猫小程序（从0-1搭建）
- 端口：会员小程序端（C端）+ B端管理端
- 核心目标：帮助会员计算购买商品可获得的积分数量，提升用户积分感知与参与度

## 功能骨架

### 一、C端小程序功能模块

#### 1. 会员信息展示模块
- 会员当前积分查询展示（对接 DL-3.2.7）
- 会员当前等级查询展示（对接 DL-3.2.4）
- 会员身份标识与积分概览卡片

#### 2. 商品列表模块
- 推荐商品列表展示（后台配置）
- 商品信息展示（图片、名称、价格、预估积分）
- 商品多选功能（支持批量选择计算）
- 点击跳转淘宝商品详情页（taobao.items.seller.list.get）

#### 3. 积分计算模块
- 已选商品汇总展示
- 积分试算接口调用（对接 DL 2.2.95）
- 计算结果展示（预估获得积分、积分明细）
- 计算历史记录（可选扩展）

#### 4. 结果展示模块
- 积分结算结果区域
- 积分获取规则说明
- 引导跳转购买或返回商品列表

### 二、B端管理端功能模块

#### 1. 多店铺兑礼活动管理
- 活动列表（查询、筛选、分页）
- 创建活动（活动名称、时间、参与店铺、规则配置）
- 启用/禁用活动
- 活动配置编辑
- 活动状态管理（未开始/进行中/已结束）

#### 2. 商品配置管理
- 商品列表管理
- 从淘宝同步商品信息（taobao.items.seller.list.get）
- 推荐商品配置（关联活动）
- 商品上下架管理

#### 3. 信息看板
- 活动曝光埋点数据展示
- 兑礼订单报表（统计、导出）
- 用户行为日志（C端用户操作异常埋点）

#### 4. 活动配置
- 活动时间配置（开始时间、结束时间）
- 推荐商品配置（选择商品、排序、展示规则）
- 活动规则配置（积分计算规则、参与条件）

### 三、系统对接模块

#### 1. 会员信息中台对接
- DL-3.2.7 会员当前积分查询
- DL-3.2.4 查询当前会员等级
- DL 2.2.95 积分试算接口

#### 2. 淘宝开放平台对接
- taobao.items.seller.list.get 批量获取商品详细信息

## 核心业务流程
```
用户进入小程序 → 展示会员信息（积分+等级）
    ↓
浏览推荐商品列表 → 选择商品（可多选）
    ↓
点击"计算积分"按钮 → 调用积分试算接口
    ↓
展示计算结果（预估获得积分明细）
    ↓
引导用户跳转淘宝购买 或 继续浏览
```

## 关键角色
- **C端会员用户**：浏览商品、选择商品、计算积分、跳转购买
- **B端运营人员**：创建活动、配置商品、查看数据报表
- **B端管理员**：活动审核、系统配置、异常监控

## 风险与关注点
- 淘宝API调用频率限制与稳定性
- 会员中台接口的响应时间与降级策略
- 多店铺场景下的数据隔离与权限控制
- 积分计算规则的准确性与实时性
