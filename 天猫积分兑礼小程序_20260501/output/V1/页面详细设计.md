# 页面详细设计文档

## 1. 首页 (/home)

### 1.1 页面说明
小程序入口页面，展示活动主题、会员积分信息和兑礼商品入口。

### 1.2 功能描述
- 展示活动背景图（可配置）
- 展示会员积分余额和等级
- 展示兑礼商品入口
- 支持跳转到商品列表、我的订单

### 1.3 业务流程图
```mermaid
graph TD
    A[页面载入] --> B[加载活动配置]
    B --> C[调用会员中台3.2.2接口]
    C --> D{接口返回成功?}
    D -->|是| E[展示会员等级和积分]
    D -->|否| F[展示默认状态]
    E --> G[展示活动背景图]
    F --> G
    G --> H[展示商品入口和订单入口]
    H --> I[结束]
```

## 2. 商品列表页 (/productList)

### 2.1 页面说明
展示所有可兑换的商品列表，支持分类筛选和排序。

### 2.2 功能描述
- 商品卡片展示（图片、名称、所需积分）
- 支持分类筛选
- 支持积分排序（从低到高/高到低）
- 支持下拉刷新和上拉加载
- 支持点击进入商品详情

### 2.3 业务流程图
```mermaid
graph TD
    A[页面载入] --> B[调用兑礼中台商品接口]
    B --> C{接口返回成功?}
    C -->|是| D[渲染商品列表]
    C -->|否| E[展示错误提示]
    D --> F[用户操作]
    E --> F
    F --> G{操作类型?}
    G -->|筛选| H[按分类筛选商品]
    G -->|排序| I[按积分排序]
    G -->|点击| J[进入商品详情]
    H --> F
    I --> F
```

## 4. 确认兑换页 (/confirmExchange)

### 4.1 页面说明
用户确认兑换信息并选择收货地址，完成积分扣减和订单创建。

### 4.2 功能描述
- 展示商品信息（缩略图、名称、所需积分）
- 选择收货地址（调用淘宝地址接口）
- 展示当前积分和兑换后剩余积分
- 确认兑换按钮
- 积分扣减和订单创建流程

### 4.3 业务流程图
```mermaid
graph TD
    A[页面载入] --> B[加载地址列表]
    B --> C[展示默认地址]
    C --> D{用户点击确认?}
    D -->|是| E[调用3.2.9积分扣减接口]
    E --> F{扣减成功?}
    F -->|是| G[创建订单]
    G --> H[订单回传会员中台]
    H --> I[跳转到兑换成功页]
    F -->|否| J[间隔2秒重试]
    J --> K{重试次数<2?}
    K -->|是| E
    K -->|否| L[订单状态=已取消]
    L --> M[跳转到兑换失败页]
    I --> N[结束]
    M --> N
    D -->|否| N
```

## 9. 活动编辑页 (/activityEdit)

### 9.1 页面说明
管理端活动创建和编辑页面，支持基础配置、活动配置、装修配置。

### 9.2 功能描述
- 基础配置：活动名称、活动说明、开始时间、结束时间
- 活动配置：关联兑礼中台活动、商品绑定
- 装修配置：背景图上传、热区配置
- 保存活动

### 9.3 业务流程图
```mermaid
graph TD
    A[页面载入] --> B{操作类型?}
    B -->|创建| C[初始化空表单]
    B -->|编辑| D[加载活动详情]
    C --> E[用户填写表单]
    D --> E
    E --> F{用户操作?}
    F -->|关联兑礼活动| G[选择兑礼中台活动]
    F -->|上传素材| H[上传背景图]
    F -->|保存| I[校验表单]
    I --> J{校验通过?}
    J -->|是| K[保存活动配置]
    J -->|否| L[展示错误提示]
    G --> E
    H --> E
    K --> M[结束]
    L --> E
```
